# coffee-backend

# ☕ Coffee Shop POS & Backend System
ระบบบริหารจัดการร้านกาแฟ และระบบขายหน้าร้าน (POS) | **สำหรับประกอบการอบรม**

**// 2/4/2569 - PROMOTIONS, RECOMMENDED MENUS & DASHBOARD** (อัปเดตล่าสุด)
* **Promotion System:** เพิ่มระบบจัดการโปรโมชั่นแบบครบวงจร รองรับการอัปโหลดรูปภาพ, ตรวจสอบวันหมดอายุ และเงื่อนไขการลดราคาตามหมวดหมู่
* **Recommended Menu:** เพิ่มระบบตั้งค่าเมนูแนะนำ และการแสดงผลแยกหมวดหมู่ในหน้า POS
* **Checkout Logic:** ปรับปรุงตรรกะการคิดเงิน ให้รองรับส่วนลดสุทธิและบันทึกลงฐานข้อมูลอย่างถูกต้อง
* **Dashboard:** เชื่อมต่อ API ดึงข้อมูลยอดขายจริงมาแสดงผลบนกราฟและตารางสรุปยอด

**// 28/3/2569 - FRONTEND**
* **POS Front-end & Cart System:** สร้างหน้า Front-end ระบบสั่งเครื่องดื่มด้วย Tailwind CSS พร้อมระบบตะกร้าสินค้า คำนวณราคารวม (รวมตัวเลือกไซส์ และท็อปปิ้งต่าง ๆ)
* **Data Fetching:** เขียนสคริปต์ดึงข้อมูลเมนูและท็อปปิ้งจากฝั่ง Back-end มาแสดงผลที่หน้า Front-end

**// 22/3/2569 - CRUD**
* **Menu Management:** สร้างโครงสร้างพื้นฐานสำหรับระบบจัดการข้อมูลเมนู (Create, Read, Update, Delete)
